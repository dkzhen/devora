## Get Started

Get started in just a few steps:

<Steps>
### 1. Create an Account

Sign up to access your Devora dashboard and get your API credentials.

[Create account →](https://devora.my.id/register)

### 2. Get Your API Key

Navigate to the API Keys section in your dashboard to generate your authentication token.

[Manage API Keys →](/docs/api-keys)

### 3. Make Your First Request

Use your API key to start making requests to any of our endpoints.

```bash
curl -X GET https://devora.my.id/api/v1/temp-mail/accounts \
  -H "Authorization: Bearer YOUR_API_KEY"
```

</Steps>
