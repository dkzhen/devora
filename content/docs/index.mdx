# Welcome to Devora Docs

Devora is an all-in-one platform for managing your tools, automation, and connected services.

> 🚀 Build faster with Devora APIs — simple, powerful, and developer-friendly.

---

## ⚡ Quick Start

<Steps>

### Get your API Key
Head over to **Tools → API Key** in your dashboard and click **Create Key**.  
Keys are prefixed with `devora_` and are linked to your account.

### Authenticate
Include your key in every API request via the `Authorization` header:

```http
Authorization: Bearer devora_your_key_here
```

</Steps>
