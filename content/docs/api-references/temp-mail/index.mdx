The Temp Mail API allows you to create and manage temporary email accounts programmatically using your API Key.

## Overview

The Temp Mail API provides a fully managed temporary email service. You can create accounts, list messages, and view email content programmatically.

For authentication details and the base URL, please visit the [API References](../api-references) page.

---

## 🚀 Quick Start

1. **Create an account** using the `/accounts` endpoint.
2. **Access the inbox** using the returned token.
3. **Manage messages** and attachments securely.
