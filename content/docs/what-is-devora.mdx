# What is Devora?

---

Devora is a unified platform designed to organize, automate, and manage tools and connected services in one place.

## Overview

Devora provides developers with a comprehensive API platform that simplifies integration, automation, and management of various services. Whether you're building a small project or an enterprise application, Devora offers the tools you need to succeed.

## What Can You Build?

Devora enables you to build a wide range of applications:

### AI-Powered Applications

Access multiple LLM providers (OpenAI, Gemini, Groq, etc.) through a single OpenAI-compatible reverse proxy endpoint. No need to manage multiple API keys or base URLs.

### Airdrop Hunting & Project Tracking

Register new projects, add tasks, and track your airdrop hunting workflow with status timelines, category filters, and task management.

### Temporary Email Services

Create and manage temporary email addresses for testing, privacy protection, and user verification workflows.

### API Management

Centralized API key management with role-based access control, usage tracking, and analytics.

### Automation Workflows

Automate repetitive tasks and integrate multiple services with our powerful API endpoints.

## Core Services

### AI Models API

Our flagship LLM proxy provides access to multiple AI models from different providers through a single unified endpoint:

- **OpenAI-compatible format** — drop-in replacement for OpenAI SDK
- **Multi-provider routing** — OpenAI, Gemini, Groq, and more
- **Streaming support** — real-time token streaming via SSE
- **Model status tracking** — know which models are active, suspended, or restricted

### Airdrop API

Programmatically manage airdrop projects and tasks:

- Register and track projects with status timelines
- Add step-by-step tasks with images and links
- Filter by status, category, and visibility
- Public project sharing (ULTRA users)

### Temp Mail API

Fully managed temporary email service:

- Instant email generation (Mail.tm & MoeMail)
- Real-time inbox monitoring with auto-refresh
- Message persistence in database
- Multiple domain support

### API Key Management

Secure and efficient API key management system with:

- Role-based access control
- Usage analytics and monitoring
- Team collaboration features
