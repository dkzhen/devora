# Sign Up

---

Create your Devora account to start using our API services.

## Create Account

Visit the registration page and fill in your details:

[Create account →](https://devora.my.id/register)

### Required Information

<Steps>
### 1. Basic Information

- **Email Address**: Your valid email address
- **Password**: Strong password (min. 8 characters)
- **Confirm Password**: Re-enter your password

### 2. Access Dashboard

Once registered, log in to access your dashboard and start using Devora services.

</Steps>

## Next Steps

After creating your account:

1. **Log in** — Access your dashboard
2. **Get API Key** — Generate your first API key
3. **Start building** — Make your first API request

<Callout type="tip" title="Pro Tip">
  Use a strong, unique password and never share your credentials with anyone.
</Callout>

Having trouble signing up? [Contact support](mailto:support@devora.my.id).
