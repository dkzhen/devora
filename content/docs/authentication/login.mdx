# Login

---

Access your Devora dashboard to manage your API keys and services.

## Sign In

Visit the login page and enter your credentials:

[Sign in →](https://devora.my.id/login)

### Login Steps

<Steps>
### 1. Enter Credentials

- **Email**: Your registered email address
- **Password**: Your account password

### 2. Access Dashboard

Once logged in, you'll be redirected to your dashboard where you can:

- View API usage statistics
- Manage API keys
- Access documentation
- Configure settings

</Steps>
